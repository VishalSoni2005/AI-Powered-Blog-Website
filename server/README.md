# 📝 Blog Website Backend - API Documentation

This repository contains the **backend API routes** for a full-stack blog website built using **Express.js**. The backend supports user authentication, blog creation and management, image uploads, user profiles, and like/comment features.

---

## 🚀 Getting Started

### Prerequisites

- Node.js
- MongoDB
- Cloudinary (for image upload, if used)
- Firebase Client ID/Secret (for social login)

### Installation

```bash
git clone https://github.com/vishalsoni2005/ai-powered-blog-website.git
cd server
npm install



PORT=3000
DB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```
